# Changelog

本文档记录 MD Editor 的版本变更。

## [0.1.13] - 2026-05-23

### 修复
- 修复图片粘贴位置和滚动问题
- 修复 Tauri 桌面版本地图片路径拼接问题
- 简化 updateStore，移除 checkedVersion 字段避免跨版本兼容问题
- 修复版本更新检测被 updateStore 时间检查阻止的问题

### 文档
- 更新 README，增加新特性说明

## [0.1.12] - 2026-05-18

### 修复
- 修复版本更新缓存问题
- 修复暗色主题代码样式

### 新增
- 新增编辑区域宽度设置（全宽、较宽、普通）

## [0.1.11] - 2026-05-17

### 新增
- 大纲栏增强：显示完整标题 tooltip 和可调整宽度

### 修复
- 移除 MSI 安装包相关代码
- 修复双击打开文件失败的问题

### 其他
- 优化 GitHub Actions 构建流程

## [0.1.10] - 2026-05-15

### 新增
- 版本检测缓存优化
- 文件关联支持（.md, .markdown）

### 修复
- 修复 EXE 版本内部拖放问题
- 修复 EXE 版本从文件目录树拖拽到窗格的问题
- 修复 EXE 版本拖放和双击顶栏问题
- 改进 EXE 版本拖放定位逻辑

## [0.1.9] - 2026-05-13

### 新增
- 分栏功能：支持水平/垂直分栏、快捷键切换和关闭窗格
- 支持从外部和文件目录树拖拽 md 文档到窗格中打开
- 支持调整分栏窗格大小

## [0.1.8] - 2026-05-12

### 新增
- 版本自动检查和更新提示

### 修复
- 修复 Vditor 资源文件缺失问题
- 修复 Vditor 本地 CDN 资源路径问题

## [0.1.7] - 2026-05-10

### 改进
- UI 优化和功能增强

## [0.1.6] - 2026-05-09

### 新增
- 最近打开文件列表
- 查找替换功能
- 模式切换快捷键（Ctrl+/）

## [0.1.5] - 2026-05-08

### 改进
- 性能优化和 UI 改进

## [0.1.4] - 2026-05-07

### 新增
- PlantUML 图表支持
- 标题栏双击最大化

### 修复
- 修复标题栏拖拽不稳定问题
- 修复 Tab 区域双击最大化无效的问题
- 修复 Workflow 竞争问题

## [0.1.3] - 2026-05-06

### 新增
- MD 文档链接跳转和嵌入功能

### 修复
- 修复 Mac 构建不生成 dmg/app 的问题
- 修复 Tauri 环境下路径解析问题

### 改进
- 优化构建配置，减少构建时间

## [0.1.2] - 2026-05-05

### 新增
- Vditor 扩展功能：上标、下标
- Mermaid 图表支持
- TOC 目录跳转

### 修复
- 修复字数统计显示问题
- 修复 TOC 点击跳转问题
- 修复 Windows 绿色版 exe 上传失败问题

## [0.1.1] - 2026-05-04

### 新增
- Windows 11 记事本风格布局
- 自定义标题栏和窗口控制

### 修复
- 修复窗口拖拽和按钮点击问题
- 修复删除文件失败问题
- 修复 Windows 路径分隔符问题

## [0.1.0] - 2026-05-03

### 初始版本
- 基于 Vditor 的 Markdown 编辑器
- 支持所见即所得、实时渲染、分屏预览三种模式
- 左侧文件目录树
- 多标签页
- 图片粘贴（Ctrl+V）
- 亮色/暗色主题切换
- 快捷键支持（Ctrl+B/I/S 等）
- Tauri 桌面应用封装
- GitHub Actions 自动构建（Windows/macOS/Linux）
